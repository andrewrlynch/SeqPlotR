# Genomic xscale for a single window

Genomic xscale for a single window

## Usage

``` r
.genomic_xscale(window_gr, w)
```

## Arguments

- window_gr:

  A `GRanges` object containing one or more windows.

- w:

  Integer index of the window to use.

## Value

Numeric length-2 vector: `c(start, end)` of the selected window.
