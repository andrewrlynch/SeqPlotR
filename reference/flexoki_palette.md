# Load colors from Kepano's Flexoki palette

Loads colors from Kepano's Flexoki palette.

## Usage

``` r
flexoki_palette(n, shade = 400)
```

## Arguments

- n:

  Number of colors to be produced. If \> 9, an equally distributed color
  ramp will be produced (and probably look bad).

- shade:

  Integer shade level. One of `200`, `300`, `400`, or `600`.

## Value

Vector of hex colors.
