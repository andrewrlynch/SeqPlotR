# Decide whether a given axis should render

Decide whether a given axis should render

## Usage

``` r
.axis_active(track, resolved_theme, side)
```

## Arguments

- track:

  The `SeqTrackR6`.

- resolved_theme:

  Track resolved theme.

- side:

  One of `"x1"`, `"x2"`, `"y1"`, `"y2"`.

## Value

Logical.
