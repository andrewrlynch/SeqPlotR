# Compute the bounding box of a track from its panel list

Compute the bounding box of a track from its panel list

## Usage

``` r
.panels_to_track_bounds(panels)
```

## Arguments

- panels:

  Output of
  [`.build_track_panels()`](http://andrewlynch.io/SeqPlotR/reference/dot-build_track_panels.md).

## Value

A list with `x0`, `x1`, `y0`, `y1`.
